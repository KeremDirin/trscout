# TRScout 🇺🇸→🇹🇷

Amerika'da yatırım almış startupları Türkiye pazarına uygunluğuna göre analiz eden AI destekli araştırma aracı.

## Özellikler
- Sektör ve yatırım aşamasına göre filtreleme
- Başlangıç sermayesi ve kurucu profili filtresi
- Türkiye uygunluk skoru (1-10)
- Pazar büyüklüğü, risk ve gelir modeli analizi

## Teknolojiler
- Node.js + Express
- Anthropic Claude API
- Vanilla HTML/CSS/JS
- Deploy: Railway
